# 🚀 ApexPlanet Web Development Internship – Task 2

## 💼 TechNova Solutions – Responsive Multi-Page Business Website

### 📖 Project Overview

TechNova Solutions is a responsive multi-page business website developed as part of **Task 2** of the ApexPlanet Web Development Internship. The project was created to improve my frontend development skills by building a modern business website with interactive features and a responsive design. The website provides a clean layout, smooth navigation, and an enhanced user experience across different devices.

---

## 🎯 Objectives

* Develop a responsive multi-page business website.
* Create a clean and professional user interface.
* Add interactive JavaScript features to improve user experience.
* Practice responsive web design techniques.
* Strengthen frontend development skills through hands-on implementation.

---
